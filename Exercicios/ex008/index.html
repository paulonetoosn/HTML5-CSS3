Texto marcado

Podemos também marcar texto usando a tag MARK

Texto grande e texto pequeno

Estamos criando um Texto grande e um texto pequeno com as tags (smol) para texto pequeno e (big) para texto grande

Texto excluido

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado com a tag(del)

Texto inserido

podemos marcar um texto como inserido para dar uma ênfase e indicar que ela foi adicionado depois com a tag (ins) (é tag semantica)

Existe também o sublihado com a tag (U) que é não (semântica)

Texto sobrescrito

Tag (sup) para inserir coisa do tipo (x20+3) X20+3

Texto subscrito

tag (sub) para inserir coisas do tipo H2O H2O
