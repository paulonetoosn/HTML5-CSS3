Vídeos em hospedagem própria

Assim ficou o código

  • A tag <<video>> é usada para definir o elemento de vídeo.
  • O atributo controls adiciona controles de reprodução (como play, pause, volume) ao vídeo.
  • O atributo width e height define a largura e altura do vídeo, respectivamente.
  • A tag <<source>> é usada para especificar as fontes do vídeo. Você pode ter várias fontes para garantir a compatibilidade com diferentes navegadores. No exemplo abaixo, o vídeo é "video.mp4", e o tipo do arquivo é especificado como "video/mp4". Você pode adicionar outras fontes para diferentes tipos de arquivos de vídeo.
  • O texto dentro da tag <<video>> é exibido caso o navegador não consiga reproduzir o vídeo.
  • Certifique-se de que o vídeo esteja localizado no mesmo diretório que seu arquivo HTML ou especifique o caminho correto para o vídeo no atributo src da tag <<source>>.
        <<video controls width="640" height="360" >>
        <<source src="video.mp4" type="video/mp4">
        Seu navegador não suporta o elemento de vídeo.

Incorporação de vídeos externos

Para incorporar vídeos externos em HTML5, você pode usar a tag <<video>> com a tag <<source>> para especificar a URL do vídeo. Aqui está um exemplo de como fazer isso: 
    <<video controls width="640" height="360">>
        <<source src="URL_DO_VIDEO" type="TIPO_DO_VIDEO">>
        Seu navegador não suporta o elemento de vídeo.
  • Substitua "URL_DO_VIDEO" pela URL do vídeo externo que você deseja incorporar e "TIPO_DO_VIDEO" pelo tipo de mídia do vídeo (por exemplo, "video/mp4" para MP4, "video/webm" para WebM, etc.).
  • Por exemplo, se você quiser incorporar um vídeo do YouTube, você pode usar o recurso de incorporação do YouTube e colocar o código fornecido dentro da tag <<source>>.
        <<video controls width="640" height="360">>
            <<source src="URL_DO_VIDEO_YOUTUBE" type="video/youtube">>
            Seu navegador não suporta o elemento de vídeo.
