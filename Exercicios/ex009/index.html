Listas

Vamos usar a tag ("ol" ordered list, listas ordenadas ) para delimitar a lista e a tag ("li" list item) para identificar cada item da lista. Existe um parametro chamado "type"que pode ser usada dentro da tag "ol" e "ul" (ex:oltype="") esse parametro vai te dar 4 opções de como você deseja ordenar sua lista. Se por acaso você deseja iniciar a classicação da lista pulando a ordem numerica ou alfabetica use o parametro (ex:ol type="--" start="--") OBS:Dentro do type para classificação ordenadas tem as opções (1, A, a, I, i,) e para não ordenadas (disc, circle, square)
