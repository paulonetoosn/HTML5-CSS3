Imagens Dinâmicas

Tente abrir esse site em vários dispositvos difrente ou simplesmente aumente e dimunua o tamanho do seu navegador

  1. A tag <<picture>> envolve as fontes de imagem alternativas e a tag <<img>>.
  2. A tag <<source>> é usada para especificar diferentes versões da imagem com base na largura da tela (media="(max-width: 600px)" para telas menores que 600px e media="(min-width: 601px)" para telas maiores ou iguais a 601px) e fornece o caminho para cada versão usando o atributo srcset.
  3. A tag <<img>> é usada como fallback para navegadores que não suportam a tag <<picture>> ou não atendem às condições especificadas nas tags <<source>>. O atributo src fornece o caminho para a imagem a ser exibida se nenhum dos <<source>> for correspondido.
  4. Com esse código, o navegador escolherá automaticamente a versão correta da imagem com base nas condições de mídia especificadas nas tags , garantindo que a imagem seja exibida de forma adequada em diferentes tamanhos de tela e dispositivos.
