Reproduzindo Áudio

Para incorporar áudio em uma página HTML5, você pode usar a tag <<audio>>.

Led Zeppelin - Since I've Been Loving You

  • A tag <<audio>> é usada para definir o elemento de áudio.
  • O atributo controls adiciona controles de reprodução (como play, pause, volume) ao áudio.
  • A tag <<source>> é usada para especificar as fontes do áudio. Você pode ter várias fontes para garantir a compatibilidade com diferentes navegadores. No exemplo acima, o áudio é "audio.mp3", e o tipo do arquivo é especificado como "audio/mpeg". Você pode adicionar outras fontes para diferentes tipos de arquivos de áudio.
  • O texto dentro da tag <<audio>> é exibido caso o navegador não consiga reproduzir o áudio.
  • Certifique-se de que o arquivo de áudio esteja localizado no mesmo diretório que seu arquivo HTML ou especifique o caminho correto para o áudio no atributo src da tag <<source>>.

Assim ficou o código

            <<audio src="Led Zeppelin - Since I've Been Loving You (Live at Madison Square Garden 1973) [Official Video].
            mp3" controls autoplay >><</audio>>
O atributo controls é utilizado na tag <<audio>> ou <<video>> em HTML para exibir controles padrão de reprodução de mídia para o usuário. Quando você inclui o atributo controls em uma tag <<audio>> ou <<video>>, o navegador automaticamente exibe uma barra de reprodução com botões de controle, como play, pause, volume, entre outros, permitindo que os usuários interajam com o conteúdo de áudio ou vídeo. Aqui está um exemplo simples: 
                <<audio controls>>
                    <<source src="audio.mp3" type="audio/mpeg">>
                    Seu navegador não suporta o elemento de áudio.
