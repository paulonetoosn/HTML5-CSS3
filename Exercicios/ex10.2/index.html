Link externo

Agora vamos usar a tag (a), conhecida como tag de âncora, é responsável por vincular os demais documentos que o nosso HTML possui, links internos e externos, dessa maneira ele vai abrir o site por cima do meu site. Para acessar O LINK Clique aqui

E para não sobrepor ao meu site, vamos usar o parametro (target="_blank:Carrega a página em uma nova janela do navegador) ("rel="external" Indica que o documento referenciado não é parte do mesmo site do documento que o referencia) assim ficou o código:(href="https://www.youtube.com/watch?v=_FKAnwaOM3g" target="_blank" rel="external">) aqui esta a musica!
